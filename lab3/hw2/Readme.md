# دليل فلاتر Django Template Language (DTL)

هذا الملف يحتوي على دليل شامل للفلاتر المدمجة في نظام قوالب Django، مصنفة حسب وظيفتها مع أمثلة عملية.

---

### النصوص والتنسيق

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `capfirst` | يحول الحرف الأول من النص إلى حرف كبير. | `{{ "django"\|capfirst }}` |
| `lower` | يحول كل أحرف النص إلى أحرف صغيرة. | `{{ "DJANGO"\|lower }}` |
| `upper` | يحول كل أحرف النص إلى أحرف كبيرة. | `{{ "django"\|upper }}` |
| `title` | يحول النص إلى تنسيق العنوان (بداية كل كلمة حرف كبير). | `{{ "django template language"\|title }}` |
| `slugify` | يحول النص إلى صيغة صديقة لمحركات البحث (URL-friendly). | `{{ "Django Template Language"\|slugify }}` |
| `truncatechars` | يختصر النص لعدد محدد من الأحرف ويضيف "...". | `{{ "Django is a web framework"\|truncatechars:10 }}` |
| `truncatewords` | يختصر النص لعدد محدد من الكلمات ويضيف "...". | `{{ "Django is a powerful web framework"\|truncatewords:4 }}` |
| `wordwrap` | يقسم النص إلى أسطر بعرض محدد من الأحرف. | `{{ "Django makes web development easy"\|wordwrap:10 }}` |
| `linebreaks` | يحول فواصل الأسطر في النص إلى وسوم `<p>` و `<br>`. | `{{ my_text\|linebreaks }}` |

### الأرقام والحسابات

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `add` | يضيف قيمة إلى المتغير (يعمل مع الأرقام والنصوص). | `{{ 5\|add:"3" }}` |
| `floatformat` | ينسق الأرقام العشرية لعدد محدد من المنازل. | `{{ 34.23234\|floatformat:2 }}` |
| `divisibleby` | يتحقق إذا كان الرقم قابلاً للقسمة على قيمة معينة. | `{{ 21\|divisibleby:"3" }}` |
| `get_digit` | يستخرج رقماً محدداً من عدد صحيح (بدءاً من اليمين). | `{{ 123456\|get_digit:3 }}` |
| `filesizeformat` | يحول قيمة عددية بالبايت إلى تنسيق مقروء (KB, MB, GB). | `{{ 123456789\|filesizeformat }}` |

### القوائم والمصفوفات

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `first` | يعرض العنصر الأول من القائمة. | `{{ my_list\|first }}` |
| `last` | يعرض العنصر الأخير من القائمة. | `{{ my_list\|last }}` |
| `join` | يدمج عناصر القائمة باستخدام فاصل محدد. | `{{ my_list\|join:", " }}` |
| `length` | يعرض عدد العناصر في القائمة. | `{{ my_list\|length }}` |
| `slice` | يعرض جزءاً (شريحة) من القائمة. | `{{ my_list\|slice:":2" }}` |
| `random` | يعرض عنصراً عشوائياً من القائمة. | `{{ my_list\|random }}` |
| `dictsort` | يرتب قائمة من القواميس تصاعدياً حسب مفتاح محدد. | `{{ users\|dictsort:"name" }}` |
| `dictsortreversed` | يرتب قائمة من القواميس تنازلياً حسب مفتاح محدد. | `{{ users\|dictsortreversed:"age" }}` |
| `unordered_list` | يحول قائمة متداخلة إلى قائمة HTML `<ul>` متداخلة. | `{{ nested_list\|unordered_list }}` |

### التواريخ والأوقات

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `date` | ينسق التاريخ باستخدام رموز محددة. | `{{ my_date\|date:"Y-m-d" }}` |
| `time` | ينسق الوقت باستخدام رموز محددة. | `{{ my_time\|time:"H:i:s" }}` |
| `timesince` | يعرض الوقت المنقضي منذ تاريخ معين بصيغة مقروءة. | `{{ past_date\|timesince }}` |
| `timeuntil` | يعرض الوقت المتبقي حتى تاريخ معين بصيغة مقروءة. | `{{ future_date\|timeuntil }}` |

### الأمان والترميز

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `escape` | يهرب رموز HTML الخاصة لحماية التطبيق من هجمات XSS. | `{{ "<b>نص</b>"\|escape }}` |
| `safe` | يمنع التهريب التلقائي للنص (يجب استخدامه بحذر). | `{{ "<b>محتوى موثوق</b>"\|safe }}` |
| `force_escape` | يقوم بالتهريب فوراً حتى لو كان النص معلم كآمن. | `{{ dangerous_var\|force_escape }}` |
| `striptags` | يزيل كل وسوم HTML من النص. | `{{ "
نص مهم
"\|striptags }}` |
| `escapejs` | يهرب الأحرف الخاصة في النص لاستخدامها بأمان في JavaScript. | `{{ "String with 'quotes'"\|escapejs }}` |
| `urlencode` | يرمز النص لاستخدامه بأمان في عناوين URL. | `{{ "django/قوالب"\|urlencode }}` |

### فلاتر `humanize` (للتنسيق البشري)

**ملاحظة:** لاستخدام هذه الفلاتر، يجب إضافة `django.contrib.humanize` إلى `INSTALLED_APPS` واستخدام `{% load humanize %}` في القالب.

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `intcomma` | يضيف فاصلة (,) للآلاف في الأرقام الكبيرة. | `{{ 45000000\|intcomma }}` |
| `intword` | يحول الأرقام الكبيرة إلى صيغة كلامية (مليون، مليار). | `{{ 1200000000\|intword }}` |
| `apnumber` | يحول الأرقام من 1 إلى 9 إلى كلمات باللغة الإنجليزية. | `{{ 1\|apnumber }}` |
| `naturalday` | يعرض التاريخ بصيغة طبيعية (اليوم، أمس، غداً). | `{{ my_date\|naturalday }}` |
| `naturaltime` | يعرض الوقت بصيغة طبيعية (منذ دقيقتين، بعد ساعة). | `{{ my_date\|naturaltime }}` |
| `ordinal` | يحول العدد إلى صيغة ترتيبية بالإنجليزية (1st, 2nd, 3rd). | `{{ 3\|ordinal }}` |

### فلاتر متنوعة

| الفلتر | الوصف | مثال |
| :--- | :--- | :--- |
| `default` | يعرض قيمة افتراضية إذا كان المتغير فارغاً أو غير موجود. | `{{ my_variable\|default:"قيمة افتراضية" }}` |
| `default_if_none` | يعرض قيمة افتراضية إذا كان المتغير `None` فقط. | `{{ my_variable\|default_if_none:"لا شيء" }}` |
| `yesno` | يعرض قيمة مخصصة لـ `True`, `False`, أو `None`. | `{{ is_active\|yesno:"نعم,لا" }}` |
| `stringformat` | ينسق سلسلة نصية وفقًا لصيغة محددة (مثل printf في C). | `{{ order_id\|stringformat:"06d" }}` |
